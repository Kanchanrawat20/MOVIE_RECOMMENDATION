Movie Recommendation System
Overview

A content-based Movie Recommendation System built using Python and Streamlit that recommends similar movies based on movie metadata and cosine similarity. The application also fetches movie posters and details using the TMDB API to provide an interactive user experience.

Features
Recommend movies similar to a selected movie
Content-based filtering using cosine similarity
Dynamic movie posters and information from TMDB API
Interactive web interface using Streamlit
Fast recommendation generation

Streamlit
